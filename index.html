 Graph-It

CS 426 Senior Project in Computer Science, Spring 2022, at UNR, CSE Department

Project Overview

    Graph-It is a graph theory toolbox and graph visualization application for students, teachers, and researchers. Drawing graphs by hand and manually running algorithms on them can be time consuming, especially if this is repeated multiple times. With Graph-It, users can create graphs, modify the graphs, and save the graphs. Users can also view information about the graph, run graph algorithms, and view the algorithm results.

Project Video


Main Project Goals

  • Develop an easy-to-use program with clear visuals that can efficiently run algorithms on graphs
  • Provide users with a clean and intuitive user interface
  • Develop a program for users to utilize without requiring coding knowlege
  • Provide intuitive features for graph creation, modification, and for running algorithms
  • Provide users the feature to save graphs in different file types
